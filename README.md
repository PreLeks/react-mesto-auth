# Фронтенд разработка проекта приложения Mesto на React с возможностью регистрациии и авторизации пользователей.
## Описание проекта и его функциональности:
Проект называется - "Mesto React Russia". С проектом можно ознакомиться [по этой ссылке](https://preleks.github.io/react-mesto-auth/#/sign-in).
Данная работа является частью одного глобального проекта (портфолио), и направлена на изучение и практическое применение навыков в HTML, CSS, JS, JSX и React, используя все возможности данных технологий. Основное направление в данном проекте - это создание сайта на основе макета из Фигмы, который должен корректно отображаться на экранах популярных размеров. В проекте использованы знания и навыки, которые необходимы для старта любого проекта данной направленности:
- Правила HTML-разметки
- Работа со стилями: расположение блоков на странице, управление цветом, шрифтами и тд.
- Флексбоксы, гриды, БЭМ и позиционирование
- Файловая структура
- Продвинутая семантика языка HTML
- Разработка интерфейсов для разных устройств с помощью медиазапросов
- Работа с относительными размерами блоков, вычисляемыми значениями, функцией calc()
- Оптимизация шрифтов под устройства с различными разрешениями: сглаживание, подгонка размера текста, рендеринг
- Работа над отимизацией изображений для более быстрой загрузки
- Изучение путей к файлам и организация файлов по БЭМ
- Создание проекта на Create React App
- Возможность создания аккаунта и редактирования данных пользователя в модальных окнах
- Возможность регистрации и авторизации пользователей
- Загрузка карточек с сервера через взаимодействие с API
- Возможность создания карточек и их удаления, отображения лайков и дизлайков визуально и с помощью счётчика также через взаимодействие с API
- Организация структуры части приложения на ref компонентах
- Организация структуры части приложения на функциональных управляемых компонентах
- Создание валидации форм в компонентах модальных окон
- Конвертация разметки в JSX
- Работа с локальным хранилищем и токеном
- Создание объекта контекста и использование провайдера
- Отправка данных и запросов в API
- Использование переменных состояния для различных компонентов
- Вынесение общего компонента для попапов, для инпутов форм и валидации

## Технологии, которые используются в проекте:
+ HTML5
+ CSS3
+ JS
+ JSX
+ Node.js
+ React
+ React Hooks
+ react-router-dom
+ API
+ REST API
